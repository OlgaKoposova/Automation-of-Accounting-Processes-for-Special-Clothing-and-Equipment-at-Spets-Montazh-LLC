# Automation-of-Accounting-Processes-for-Special-Clothing-and-Equipment-at-Spets-Montazh-LLC
# Project Activity
Role: 1С Analyst
Duration: 8 months
Objective: To optimize the accounting processes for special clothing and equipment at a mid-sized industrial enterprise, reduce manual labor costs, and improve transparency of document flow.
# Problem Statement
The company faced the following challenges:
Manual tracking of special clothing issuance led to frequent errors
No unified system for write-offs and returns
Difficulty tracking expiration dates and re-issuance deadlines
New employees struggled to understand the existing processes
# My Contributions
Modeled the "as-is" and "to-be" business processes using BPMN notation
Collected and structured requirements (business/user, functional/non-functional)
Wrote technical specifications (TZ) for 1С developers
Created PMI (Program and Methodology of Testing) and test cases
Developed user instructions for the updated system
# Tools & Technologies
BPMN (business process modeling)
1С: Enterprise platform
Requirements tracking (registry)
Test documentation (test cases, bug reports)
# Results
Reduced time spent on routine accounting tasks by an estimated 30%
Improved inter-departmental communication through clear requirement documentation
Lowered the entry barrier for new users with detailed instructions
# What I Learned
How to translate business needs into technical tasks
The importance of separating functional and non-functional requirements
Practical testing strategies for 1С configurations
